# Verifiable Bitcoin Accounts

Verifiable Bitcoin Accounts (VBA) let institutions deploy Bitcoin into onchain lending and yield markets without changing custody, introducing a new counterparty, or compromising on legal protections.&#x20;



Built on Threshold Network's signer infrastructure, every spending condition, recovery path, and migration rule is encoded in Bitcoin Script and verifiable onchain.
